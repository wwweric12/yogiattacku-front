# 🌏 여기어때유

## 1. 프로젝트 개요
**여기어때유**는 사용자의 취향과 여행 목적에 맞는 **맞춤형 여행지 추천**을 제공하는 **AI 챗봇 서비스**입니다.  
OpenAI 또는 Claude의 언어 모델을 기반으로 자연스러운 대화를 통해 여행지를 추천받고,  
사용자는 추천받은 장소에 대한 후기, 블로그, 사진을 직접 공유할 수 있습니다.  




## 2. 주요 기능

### 로그인/회원가입

- KAKAO OAUTH로 진행
    - accessToken body값으로
    - refreshToken cookie로

- 401이면 쿠키 확인해서 refreshToken가지고 accessToken 다시 요청해서 
accessToken을 받으면 다시 요청
- **해당 부분을 어떻게 처리할지 고민 해봐야함**

해당 서비스는 무조건 로그인을 해야 이용할 수 있음


## 헤더

- 로고 (왼쪽)
    - 클릭시 메인 페이지로 이동
- 챗봇(오른쪽)
    - 챗봇 클릭시 챗봇 페이지로 이동
- 게시판(오른쪽)
    - 게시판 으로 이동
- 마이페이지(오른쪽)
    - 마이페이지로 이동
- 로그아웃
    - 로그인을 한경우 로그아웃으로 표시 클릭시 로그아웃
- 프로필 사진
    - 로그인 한경우에만 존재
